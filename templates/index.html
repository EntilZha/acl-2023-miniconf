Welcome to {{ config.display_name}}!

{{ config.display_name }} is hosted on a combination of underline.io, RocketChat, and MiniConf (this site)!

Program
Currently, only the program for papers is on this site, which includes: main track papers (orals + in-person posters + virtual posters + spotlight), demo papers, and industry papers.
How can I find my session?
  1. Click "Papers".
  2. Search for your paper.
  3. Click on the title.
  4. Check the session assignment and time.
Volunteer
We are actively working on getting this site ready for ACL 2023, if you want to help, open an issue to volunteer.

Schedule
Use the Schedule to find live sessions and set your timezone.
Chat
Chat with attendees using various RocketChat channels! When you access RocketChat for the first time, click on 'Login with Underline' and your Underline credentials will be automatically used to log you in to RocketChat.
